Doctor at Your Fingertip

This was a class project I worked on as a group while taking EE 474. Its functionality include:

  1. Measurement of
    1. Temperature
    2. Blood Pressure
    3. Pulse Rate
    4. Respiration Rate
    5. EKG
  2. Computation of
    1. F to C
    2. mmHg
    3. Hz to BPM
    4. Fourier Transformation
  3. Alarm
  4. Remote Control (via PuTTY)

Which has been built using Arduino MEGA 2560 and Uno. Code was written in C, available here: https://github.com/a1m0le/ee474-A2
